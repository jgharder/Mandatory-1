Variables

We have 3 different variables in JavaScript 

    var
    let
    const
Var is generally not used anymore, as it was a global variable. And not scope dependent. This could create problems, if you were not 100% sure of what you are doing. Let is used to declare variables at block scope. Let is used when the variable is going to be reassigned
Const is used when you want to declare a variable that is not going to be reassigned. It's also block scope.
Use const as a default, when declaring variables.


Spread operater (...)

Spread operater is used when you want to copy or insert new data in to an existing variable. 

    [...iterableObj, '4', 'five', 6]; 
    // combine two arrays by inserting all elements from iterableObj
